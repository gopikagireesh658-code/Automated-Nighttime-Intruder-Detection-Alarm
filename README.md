# Automated-Nighttime-Intruder-Detection-Alarm
This Arduino Smart Night-Security System uses an LDR and PIR sensor to protect property after dark. It stays idle by day to save energy. If it senses movement in the dark, it triggers a multi-stage defense: flooding the area with 5 synchronized LEDs to startle intruders, and sounding a high-urgency dual-tone siren to alert homeowners.
